## What you can do in Loupe Factory

<div class="lf-feature-grid">
  <a href="platform-guide/inventory-management/" class="lf-feature-card">

  <a href="platform-guide/production-tracking/" class="lf-feature-card">

  <a href="platform-guide/order-management/" class="lf-feature-card">

      each order through fulfillment.
    </div>
  </a>

  <a href="platform-guide/shipment-tracking/" class="lf-feature-card">

    <div class="lf-feature-card__desc">

      creation through delivery.
    </div>
  </a>

    </div>
  </a>
