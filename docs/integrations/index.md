## How to use this section

1. Start with the guide for the system you want to connect.
2. Complete the **Before you begin** checklist in that guide.
3. Run a small test sync or import before you use the integration for daily
   work.
4. Review the mapping tables so your team knows what each system is responsible
   for.

<div class="lf-guide-note">
  <p>
    <strong>Good rollout practice:</strong> connect one integration at a time,
    test with a small sample, then train the team on the exact review steps
    they should follow after each sync or import.
  </p>
</div>
